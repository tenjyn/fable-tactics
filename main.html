Fable Tactics

Welcome to Fable Tactics, a stat-based take on chess. Use this page to get set up and learn the basics before diving into a match.

Setup

  1. Visit the Dashboard to choose a piece type and view its stats.
  2. Press Start Game on the dashboard to launch the board.
  3. Play the game on the Game Board.

How it Works
